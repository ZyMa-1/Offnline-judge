This site contains an archive of Olympiad programming tasks with a built-in testing system. To sumbit problem go to practice section, where currently more than 2 problems, paste your code and click on submit button. After submitting system test your solutions on tests and then returns verdict.

Testing system uses the following compilers:


g++ -static -std=c++11 -o submission.exe submission.cpp 




Status
 
Event
 
AC
 
Accepted. Program works correctly and passed all tests
 
CE
 
Compilation error. Syntax error in the program or the file extension is specified incorrectly
 
WA
 
Wrong answer. The result of the program does not match with correct answer
 
TLE
 
Time limit exceeded. Program runs longer than the installed time
 
MLE
 
Memory limit exceeded. Program uses more than the installed memory size
 
RE
 
Runtime error. Program terminated with a non-zero return code.
Mostly causes by array overflow or division by zero
